# PZEM-016andRPi-PowerMon
Mains power monitor using PZEM-016 and R Pi

# Status

Being built up based on https://github.com/grayerbeard/energyMaster and the log Python used on other projects.

The aim is a system that 
* monitors power use
* sends alerts by email when power use unusual
* sends regular reports by email of power use
* generates a web page showing status and recent power use that can be consulted localy or remotely
